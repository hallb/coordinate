## Working with AI

When using these documents with an AI assistant:

- Fill in each document incrementally through conversation with the AI.
- Reference specific documents by name so the AI can focus context.
- Use the glossary to establish shared terminology early.
- Revisit and refine documents as understanding deepens -- requirements are living artifacts.
